This project is an introduction in implementing interactivity in Jupyter Notebooks (.ipynb).
It introduces some of the packages like `ipywidgets` or `ipyleaflet` and shows their basic functionality.
In the future I want to build on this knowledge and create interactive dasboards in Jupyter.
